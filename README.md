# KSPCR
KSPCR is a fantasy computer. The instructions are below and a sample TXT project demos will be in the repository.
# INSTRUCTIONS
Set I - 1/
Replace RAM - 2/
Go to X - 3/
Go to Y - 4/
Draw a part of ram at the X and Y position - 5/
Jump to line - 6/
Jumps to line (item 1 of RAM) if specified section of RAM is equal to I - 7/
Adds the specified number and I together - 8/
Halt (STOP ALL)- 9/
Gets the key the user is pressing - 10/
clear the screen - 11/
